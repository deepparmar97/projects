About
---------
The project beatsolve is a fun quiz based website which involves the user to solve questions. After solving the questions 
and selecting the aprropriate answer the user unlocks an audio thats a part of piano sample.Then on the second level the 
user unlocks a drum loop sample and then a bass sample.After all the levels the user comes on the final screen where all 
the samples play together and an animation displays.

Whats Behind
---------------
The whole project is solely based upon javascript , HTML and CSS and 
does not incorporate any other techology.There is use of arrays,
functions,constructors and other several things under javascript.

How to use
-----------
Open index.html on a browser and answer the questions and ENJOY!!
